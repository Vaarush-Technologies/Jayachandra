My Projects

Data Encryption Using Matrices

Data encryption using matrices involves using a matrix as a cipher to encrypt a message. The steps for encrypting a message using matrices are:

  1. Assign numbers to letters: Map each letter in the message to a number.
  2. Convert to vectors: Convert the sequence of numbers into vectors of a specific size, such as 3x1.
  3. Join vectors: Combine the vectors to form a message matrix.
  4. Multiply by encoding matrix: Multiply the message matrix by an encoding matrix.
  5. Convert to sequence: Convert the encoded message matrix back into a sequence of numbers to transmit.
  6. The key matrix used to encrypt messages must be kept secret between the sender and recipient. If the key matrix is discovered, the intercepted messages can be decoded.
